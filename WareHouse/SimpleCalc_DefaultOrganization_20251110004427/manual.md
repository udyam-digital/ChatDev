# Simple Calculator

A simple GUI calculator that can perform addition, subtraction, multiplication, and division of two numbers.

## Main Functions

The Simple Calculator provides the following functionalities:

- **Addition**: Adds two numbers.
- **Subtraction**: Subtracts the second number from the first number.
- **Multiplication**: Multiplies two numbers.
- **Division**: Divides the first number by the second number (with error handling for division by zero).
- **Clear**: Resets the input fields and result display.

## Installation

To run the Simple Calculator, you need to have Python and the Tkinter library installed. Tkinter is included with most Python installations, but if you need to install it separately, you can do so using the following commands:

1. **Install Python**: Download and install Python from [python.org](https://www.python.org/downloads/).

2. **Install Tkinter**: If Tkinter is not already installed, you can install it using pip:
   ```bash
   pip install tk
   ```

## How to Use the Calculator

1. **Run the Application**:
   - Open a terminal or command prompt.
   - Navigate to the directory where `main.py` is located.
   - Run the following command:
     ```bash
     python main.py
     ```

2. **Using the Calculator**:
   - Enter the first number in the "First Number" input field.
   - Enter the second number in the "Second Number" input field.
   - Click on one of the operation buttons (Add, Subtract, Multiply, Divide) to perform the desired calculation.
   - The result will be displayed below the buttons.
   - If you wish to clear the input fields and result, click the "Clear" button.

3. **Error Handling**:
   - If you enter invalid numbers (non-numeric values), an error message will be displayed prompting you to enter valid numbers.
   - If you attempt to divide by zero, an error message will inform you that division by zero is not allowed.

## Example Usage

1. To add 5 and 3:
   - Enter `5` in the "First Number" field.
   - Enter `3` in the "Second Number" field.
   - Click the "Add" button.
   - The result displayed will be `Result: 8.0`.

2. To divide 10 by 0:
   - Enter `10` in the "First Number" field.
   - Enter `0` in the "Second Number" field.
   - Click the "Divide" button.
   - An error message will appear stating "Cannot divide by zero".

## Conclusion

The Simple Calculator is a straightforward application designed to perform basic arithmetic operations. It is user-friendly and provides immediate feedback for both valid and invalid inputs. Enjoy using the calculator!
